Formsend

Formsend

描述 : 表单验证

作者 : ywchen(陈余文)

类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.form String [必填] 要验证提交的Form；opts.valiOpts.form和opts.form必须有一个存在值；当两者都存在值，但不一致时，以opts.valiOpts.form为准；
opts.url String [必填] 表单提交AJAX的URL；和opts.goOpts.url是一样的；
opts.data String 表单提交AJAX的额外数据；和opts.goOpts.data是一样的；
opts.successText String 验证成功时显示的文字,在默认opts.validationOpts.success不被修改时有效
opts.valiOpts String Validation组件的参数，参见Validation组件说明;opts.valiOpts.*的属性都可以opts.*方式配置，如果有相同，以opts.valiOpts为优先
opts.goOpts String Go组件的参数，参见GO组件说明

返回值

类型 说明
Formsend Formsend的实例

示例

			


            var frmSend = new Formsend({

				form:"#frmInfo"

				,"url":"/"

				,"successText":"验证成功了耶"

				,goOpts:{

					"success":function(){

						console.log("GO success");

					}

					,"fail":function(){

						console.log("GO fail");

					}

				}

			});
