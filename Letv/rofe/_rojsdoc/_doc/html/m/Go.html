类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.url Function [必填] 提交数据的URL
opts.success Function 表单验证单控件成功时的回调,参数为AJAX的数据
opts.fail Function 表单验证单控件失败时的回调，参数为什么类型的错误(ajaxError,customError)
opts.lock Object 防重复提交DOM,一般是被点击的按钮
opts.isPush Object 是否以队列方式提交 --- debug --- 未实现
opts. Object 其它$.ajax的配置参数

类属性方法

方法名称 说明 参数
abort 中断本次AJAX请求 无
addLock 添加LOCK状态 无
removeLock 移除LOCK状态 无
isLock 检查是否被LOCK 无
setSetup 设置全局 AJAX 默认选项,和$.ajaxSetup的使用方式一致 无
