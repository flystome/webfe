类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.container String [必填] 要验证提交的Form；opts.valiOpts.form和opts.form必须有一个存在值；当两者都存在值，但不一致时，以opts.valiOpts.form为准；
opts.url String [必填] 表单提交AJAX的URL；和opts.goOpts.url是一样的；
opts.query String opts.container容器中哪些可以做用提交数据项，例："input,textarea",这些标签得含有Name属性
opts.data String 表单提交AJAX的额外数据；和opts.goOpts.data是一样的；
opts.sucShowMsgTime String 成功后显示成功MSG的时间，前提是isAutoHideTip被设置为false;默认1500毫秒
opts.hasTips boolean 是否有Tip显示,默认为true(显示)
opts.isAutoHideTip boolean 是否在请求回来后自动隐藏tip，默认true(隐藏)
opts.successText String 验证成功时显示的文字,在默认opts.validationOpts.success不被修改时有效
opts.valiOpts String Validation组件的参数，参见Validation组件说明;opts.valiOpts.*的属性都可以opts.*方式配置，如果有相同，以opts.valiOpts为优先
opts.goOpts String Go组件的参数，参见GO组件说明
opts.tipsOpts String Tips组件的参数，参见Tips组件说明
opts.getData function 获取最终提交的数据

类属性方法

方法名称 说明 参数
showTips 设置Tip为显示状态 无
hideTips 设置Tip为隐藏状态 无
setTipStyle 重置Tips位置为最佳显示位置；也可设置Tips的宽度
参数名称 参数类型 说明
width Number Tips的宽度,可选参数
setTipContent 重置Tips位置为最佳显示位置；也可设置Tips的宽度
参数名称 参数类型 说明
content String 重置Tips的内容
