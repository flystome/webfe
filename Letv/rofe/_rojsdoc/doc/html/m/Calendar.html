Calendar

Calendar

描述 : 日期组件 * this.opt.date和this.opt.input里的时间，优先显示this.opt.input里的时间；

作者 : ywchen(陈余文)

demo

类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.container HTMLElement [必填] 日期容器
opts.input HTMLElement [必填] 获取被选中日期的Dom
opts.weekOffset Number 星期日的列排列顺序，传值被限制在-6到0之间，含0和-6；0时，在第一列，-1是在第二列，-2是在第三列，以此类推;当值大于0时按0算，当值小于-6时按-6计算,默认值0
opts.date Date 初始化的日期
opts.fromDate Date 日期可激活区间，开始区间
opts.toDate Date 日期可激活区间，结束区间
opts.titleFormat String 不可以带时分秒，默认值yyyy-MM-dd
opts.format String 日期输出格式代，默认值yyyy-MM-dd
opts.cellTpl String 时间格子的模板，默认值是'<span class="{className}" title="{title}" data-value="{dateStr}">{date}</span>'，其中{title}为按format格式的数据，{date}为单纯的日，{className}为相应的样式，{dateStr}是标准的输出格式yyyy/MM/dd hh:mm:ss
opts.cssClearFloat String 清除浮动样式，默认值clearx
opts.cssAllClass String 每个格式都有的样式，默认值ro_date_cell
opts.unCurMonth String 非当前月的日期的格子样式，默认值ro_date_hui
opts.headCell String 星期的头部格子样式，默认值ro_date_head
opts.footCell String 尾部的格子样式，默认值ro_date_foot
opts.cssEnter String 最后一个格子的样式，默认值ro_date_enter
opts.htmlEnter String 最后一个格子后插入的Dom，默认值<br class="ro_date_br">
opts.cssLine String 将格子变成一整行的样式，默认值ro_date_line
opts.cssDisabled String 不在可选中区间的日期格子样式，默认值ro_date_disabled
opts.cssSelect String 日期格子被选中时的样式，默认值ro_date_select
opts.max Number 最大输出的格子个数,暂时只支持42;默认值42
opts.hasHour Boolean 是否显示小时
opts.hasMinutes Boolean 是否显示分钟
opts.hasSeconds Boolean 是否显示秒
opts.isReject Boolean 是否存在显示排斥，一般是弹框的容器里会出现这样的需要
opts.callback function 选中是的回调
opts.unAreaFail function 当前要选中的时间不在可选区间中
opts.unSelectDateFail function 日期还未选中，不能直接修改时分秒
opts.show function 时间控件容器被显示回调
opts.hide function 时间控件容器被隐藏回调

类属性方法

方法名称 说明 参数
init 组件初始化入口，初始化指定的年月的当月日期；没有指定时以参数this.opt.date为准；当this.opt.date没有时以当前日期为准
参数名称 参数类型 说明
year Number 初始化日期的年,可不值就是今天的年月
month Number 初始化日期的月,可不值就是今天的年月
show 日期组件显示 无
hide 日期组件隐藏 无
getInputValue 获取日期输入框Dom的值 无
setInputValue 设置日期输入框Dom的值
参数名称 参数类型 说明
value String 日期格式
getDateDom 获取指定日期的格子Dom
参数名称 参数类型 说明
dateStr String 日期格式
format 日期格式化函数
参数名称 参数类型 说明
date String|Number|Date 日期,可日期格式的字符串、可毫秒的数值1433835125037、可时间对象
format String 格式化格式
getObjectCalendar 不管日期是字符的还是对象的，最终能返回一个对象的日期，返回的时间可能是NULL
参数名称 参数类型 说明
date String|Number|Date 日期,可日期格式的字符串、可毫秒的数值1433835125037、可时间对象
getObjectDate 不管日期是字符的还是对象的，最终能返回一个对象的日期，肯定能返回一个时间
参数名称 参数类型 说明
date String|Number|Date 日期,可日期格式的字符串、可毫秒的数值1433835125037、可时间对象
renderMonth 渲染组件的主函数模板
参数名称 参数类型 说明
year Number 初始化日期的年
month Number 初始化日期的月
goReload 调用关联的此Calendar实例的另外Calendar实例重载
参数名称 参数类型 说明
date String|Number|Date 日期,可日期格式的字符串、可毫秒的数值1433835125037、可时间对象
reload 当本身有日期选中的时候，要修改关联的对象的可选中状态
参数名称 参数类型 说明
opt Object [必填] 参数
setMinDate 设置最小时间的对象
参数名称 参数类型 说明
arg_calendar Calendar 关联的Calendar对象的实例
setMaxDate 设置最大时间的对象
参数名称 参数类型 说明
arg_calendar Calendar 关联的Calendar对象的实例
prevMonth 初始化上个月显示 无
nextMonth 初始化下个月显示 无
prevYear 初始化上个年显示 无
nextYear 初始化下个年显示 无

返回值

类型 说明
Calendar Calendar的实例

            calendar1 = new Calendar({

                container:".J_list1"

                ,input:"#txtInput1"

                ,titleFormat:"yyyy年MM月dd日"

                ,format:"yyyy-MM-dd hh:mm:ss"

                ,callback:function(dateStr,dom){

                    //this.hide();

                    //log(arguments)

                }

                ,hasHour:true

                ,hasMinutes:true

                ,hasSeconds:true



                ,date:"2015/05/01"

                ,fromDate:"2015/04/01"

                ,toDate:"2015/06/23"





            });



            calendar2 = new Calendar({

                container:".J_list2"

                ,input:"#txtInput2"

                ,titleFormat:"yyyy年MM月dd日"

                ,format:"yyyy-MM-dd"

                ,callback:function(dateStr,dom){

                    //$("#txtInput").val(dateStr);

                    log(arguments)

                }

                ,hasHour:false

                ,hasMinutes:false

                ,hasSeconds:false



                //,date:"2015/05/01"

                //,fromDate:"2015/04/01"

                //,toDate:"2015/05/23"





            });



            calendar1.setMaxDate(calendar2);

            calendar2.setMinDate(calendar1);
