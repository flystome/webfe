类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.container HTMLElement [必填] 被缩放的容器
opts.img Object 图片Dom,必须是container容器的子Dom；查找方式为container.find(img),默认值是”img“
opts.plus HTMLElement 放大的按钮DOM
opts.subtract HTMLElement 缩小的按钮DOM
opts.view HTMLElement 恢复初始按钮DOM
opts.levelDom HTMLElement 等级显示容器
opts.progressContainer HTMLElement 进行条容器
opts.progressDom HTMLElement 进行条容器进度的容器
opts.dotDom HTMLElement 进行条容器上的点
opts.maxLevel Number 最大放大等级，默认10级
opts.offsetX Number X偏移值
opts.speed Number 每次放大或缩小的数值；默认为”10“px;以宽为参考
opts.cutWidth Number 裁后的最终宽；默认是0，即按图片显示的大小，当为0或null是都是图片显示的大小；它必须是cutWidth/cutHeight = $(opts.container).width() / $(opts.container).height(); getData的数据会根据cutWidth / $(opts.container).width()的比例，计算出其它值
opts.cutHeight Number 裁后的最终高；默认是0，即按图片显示的大小，当为0或null是都是图片显示的大小；它必须是cutWidth/cutHeight = $(opts.container).width() / $(opts.container).height(); getData的数据会根据cutWidth / $(opts.container).width()的比例，计算出其它值
opts.load HTMLElement 控件的图片加载中
opts.success HTMLElement 控件初始化成功 OR 图片加载成功
opts.fail HTMLElement 控件初始化失败 OR 图片加载失败

类属性方法

方法名称 说明 参数
zoom 图片的放大或缩小接口
参数名称 参数类型 说明
type String|Number 取值范围 “+”，放大；“-”缩小；其它仅能为数值型，即放大的级别，可为小数
getData 获取相关数据，如原始图片大小(width,height)、当前的图片大小(width,height)、当前的显示起点(x,y) 无
scale 设置显示级别
参数名称 参数类型 说明
level Number 放大级别，可为小数
getImgDom 获取图片Dom 无
initView 初始化视图到最小级别 无
loadImg 初始化视图到最小级别
参数名称 参数类型 说明
imgUrl String 图片地址
callback Function 回调 * 参数:①img对象{object}，②成功或失败的标识{boolean},true为成功
getCenter 获取可视区域中心点 无

返回值

类型 说明
Cutpicture Cutpicture的实例

示例

			


            new Cutpicture({

                container:".picBar"             // 控件容器Dom

                ,plus:".J_ZoomUp"               // +按钮

                ,subtract:".J_ZoomDown"         // -按钮

                ,view:".J_base"                 // 恢复初始按钮

                ,maxLevel:30                    // 放大等级，默认10级

                ,progressContainer:$(".pro")    // 进行条容器

                ,progressDom:$(".l")            // 进行条容器进度的容器

                ,dotDom:$(".dot")               // 进行条容器上的点

                ,offsetX:6                      // 偏移值

                ,speed:40                       // 每次放大或缩小的数值；默认为10px;以宽为参考；

                ,cutWidth:1500

                ,cutHeight:1515

            });
