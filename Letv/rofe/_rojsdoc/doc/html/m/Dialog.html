类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.content String|Function|HTMLElement [必填] 对话框的内容
opts.width Number 对话框宽度，默认300px
opts.height Number 对话框高度，默认自动
opts.isStop Boolean 对话框是否不跟随滚动，默认为不跟随
opts.timer Number 设置自动关闭时间，毫秒级，默认不自动关闭
opts.hasMask Boolean 是否显示蒙板，默认显示
opts.classes Boolean 配置样式{"iframe":"","window":"","mask":""}
opts.idsPrefix Boolean 配置样式ID前缀{"iframe":"","window":"","mask":""}
opts.init Function 对话框被初始化时的回调，仅触发一次
opts.confirm Function 点击确定按钮时被回调
opts.colse Function 点击关闭按钮时被回调，和ucolse的区别是：colse被定义为程序处理的，ucolse被用户主动发起的
opts.ucolse Function 点击关闭按钮时被回调，和colse的区别是：colse被定义为程序处理的，ucolse被用户主动发起的
opts.beforeOpen Function 对话框被打开显示前的回调
opts.afterOpen Function 对话框被打开显示后的回调
opts.beforeClose Function 对话框被关闭或被移除前的回调
opts.afterClose Function 对话框被关闭或被移除后的回调
opts.cancel Function 只要对话框被移除或隐藏都被回调；回调的参数恒等于true是，是用户主动关闭的

getMaskDom 获取对话框的蒙板 无
getDialogDom 获取对话的框Dom 无
resize 重新设置定位 无
closeOther 关闭其它窗体，慎用
参数名称 参数类型 说明
