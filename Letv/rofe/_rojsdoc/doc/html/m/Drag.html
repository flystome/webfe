描述 : 拖拽类

作者 : ywchen(陈余文)

demo

类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.drag HTMLElement [必填] 可拖拽对象
opts.box HTMLElement 被拖拽对象（被拖拽对象包含可拖拽对象）
opts.area HTMLElement 可拖拽区域容器
opts.isEnable Boolean 是否启用拖拽
opts.limitAreaOffset HTMLElement 可拖拽区域容器的外加限制：{x:10,y:20}
opts.limitAreaSize HTMLElement 可拖拽区域容器的外加限制：{width:10,height:20}
opts.scrollTarget HTMLElement 滚动条滚动目标
opts.scrollOffset Object|Function 滚动条的偏移量
opts.offset Object|Function 偏移量

类属性方法

方法名称 说明 参数
forceDown 强制开启down
参数名称 参数类型 说明
Event [必填] e 被启动mousedown事件传递过来的event

返回值

类型 说明
Dialog Dialog的实例

示例

			

            new Drag({
                isEnable:true,
                ,drag:$(".title")
                ,box:$(".dialog")
                ,area:$("body")
                ,offset:function(){
                    return {x:0,y:0}
                } || {x:0,y:0}
            });
