类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.btn String|HTMLElement [必填] 触发出列表的按钮Dom
opts.list String|HTMLElement [必填] 列表Dom
opts.btnOpenEvent String 按钮Dom触发显示事件，默认为mouseenter
opts.btnCancelEvent String 按钮Dom触发隐藏事件，默认为mouseleave；--- debug --- 暂不支持btnOpenEvent和btnCancelEvent事件都click的时候，即点一下显示再点一下隐藏的情况
opts.listOpenEvent String 列表Dom触发显示事件，默认为mouseenter
opts.listCancelEvent String 列表Dom触发隐藏事件，默认为mouseleave
opts.init Function 组件被初始化完后回调
opts.beforeOpen Function 列表Dom被显示前回调；第一次参数是最后一次触发的按钮，第二个参数是被显示列表
opts.open Function 列表Dom被显示时回调；第一次参数是最后一次触发的按钮，第二个参数是被显示列表
opts.beforeCancel Function 列表Dom被隐藏前回调；第一次参数是最后一次触发的按钮，第二个参数是被隐藏列表
opts.cancel Function 列表Dom被隐藏时回调；第一次参数是最后一次触发的按钮，第二个参数是被隐藏列表
opts.time Function 事件延迟器时间，默认200毫秒

类属性方法

方法名称 说明 参数
hide 强制列表被隐藏,并触发隐藏回调 无
getCurrBtnElement 获取当前触发显示列表的按钮，无列表显示时，返回值为null 无
getBtnElements 获取按钮列表Dom 无
getListElements 获取下拉列表Dom 无

返回值

类型 说明
Dropdown Dropdown的实例

示例

			

			var dropdown = new Dropdown({
				btn:".J_btn"
				,list:".J_list"
				,btnOpenEvent:"mouseenter click"
				,btnCancelEvent:"mouseleave"
				,listOpenEvent:"mouseenter"
				,listCancelEvent:"mouseleave"
				,time:200
				,open:function(){
					var currBtnElement = this.getCurrBtnElement();
					$(currBtnElement).addClass("bgRed");
				}
				,cancel:function(){
					var currBtnElement = this.getCurrBtnElement();
					$(currBtnElement).removeClass("bgRed");
				}
			});
