Page

描述 : 翻页组件，模板中的{href}这种形式的都是变量；替换模板时应尽量保留以保证功能完整性

作者 : ywchen(陈余文)

demo

类说明

类参数

参数名称 参数类型 说明
opt Object 参数
opt.container HTMLElement [必填] 组件容器
opt.length Number [必填] 总记录数，默认0条
opt.curPage Number 当前页码,默认第一页
opt.count Number 每页的条数,可选参数，默认为10
opt.size Number 每屏的页码数,可选参数，默认为10
opt.hasFirstAndEndTag Boolean 是否有首页和末页标签，默认true,显示
opt.hasPreAndNextTag Boolean 是否有上一页和下一页标签，默认true,显示
opt.hasJumpTag Boolean 是否有跳转到标签，默认true,显示
opt.jumpTpl String 跳转到的模板，输入框必须带J_roPageInput样式名；按钮必须带J_roPageButton样式名；跳转到的模板；默认为‘<input class="ui-text J_roPageInput" type="text" placeholder="页码"><span class="ui-button J_roPageButton">跳转</span>’
opt.linkTpl String 页码的模板，默认为'<a title="{text}" href="{href}" class="{linkClass} J_roPageLink" data-ro-page={page}>{text}</a>'
opt.btnTpl String 首页、末页、上页、下页按钮的模板,默认'<a title="{text}" href="{href}" class="{linkClass} J_roPageLink" data-ro-page={page}>{text}</a>'
opt.parentTpl String 父节点的容器模板,格式如‘<div>{html}</div>’；默认空
opt.firstText String 首页按钮的文本，默认”首页“
opt.endText String 末页按钮的文本，默认”末页“
opt.preText String 上一页按钮的文本，默认”上一页“
opt.nextText String 下一页按钮的文本，默认”下一页“
opt.linkHref String 模板中变量参数为{href}的值
opt.midText String ”页码“标签的文本内容，默认{page}
opt.commonClass String ”首页“、”末页“、”上一页“、”下一页“以及页数标签的能用样式
opt.curClass String 选中的样式，默认cur
opt.preClass String ”上一页“的样式，无默认
opt.nextClass String ”上一页“的样式，无默认
opt.firstClass String ”首页“的样式，无默认
opt.endClass String ”末页“的样式，无默认
opt.btnDisableClass String ”首页“、”末页“、”上一页“、”下一页“的不可用状态样式；当存在值时，当前是第一页也会出现”首页“等
opt.maxPage Number 最大页码,可选参数，默认为0,不限制
opt.jumpParamError function 当输入的内容不符合时被回调，该函数返回值是true,继续跳转；若为false不跳转；函数参数，第一个是当前页，第二个是最大页
opt.beforeJump function 当页码跳转前被回调，该函数返回值是true,继续跳转；若为false不跳转；函数参数，第一个是当前页，第二个是最大页
opt.rendered function 当页码跳转后且渲染完后被回调，该函数返回值是true,继续跳转；若为false不跳转；函数参数，第一个是当前页，第二个是最大页

类属性方法

方法名称 说明 参数
getContainer 获取组件的容器 无
getPage 获取上次渲染时的页码 无
goToPage 跳转到指定的页码
参数名称 参数类型 说明
curPage Number 被跳转到的页码, 当参数不合理时还是会跳转；比如大于最大页码时按最大页码跳转
getFirstDom 获取”首页“的Dom 无
getEndDom 获取”末页“的Dom 无
getPreDom 获取”上一页“的Dom 无
getNextDom 获取”下一页“的Dom 无

返回值

类型 说明
Page opt.返回Page的实例
