类说明

类参数

参数名称 参数类型 说明
opts Object 参数
opts.target String [必填] 目标Dom，即被Tip的小三角指向的
opts.content String 内容
opts.zIndex String 组件DOM的Sytle.z-index值,默认1001
opts.width String 组件宽度,默认300
opts.parent String 放置当前Tip的Dom的容器
opts.hasCloseBtn String 是否有"X"关闭按钮,默认有X关闭功能
opts.offset Object 偏移值
参数名称 参数类型 说明
opts.offset.x String 横坐标上的偏移值，默认值0
opts.offset.y String 纵坐标上的偏移值，默认值0
opts.classes Object 样式配置
opts.show function 组件被显示时的回调
opts.cancel function 组件被关闭时的回调

类属性方法

方法名称 说明 参数
setStyle 设置显示的位置和样式
参数名称 参数类型 说明
width Number Tips的宽度
setDirection 设置方向
参数名称 参数类型 说明
type String 箭头上下左右的样式名
getDom 获取当前Tip的Dom 无
setContent 设置Tip显示的内容
参数名称 参数类型 说明
value String 显示的内容
show 设置Tip为显示状态
参数名称 参数类型 说明
width Number Tips的宽度,可选参数
hide 设置Tip为隐藏状态 无
close 设置Tip为隐藏状态且移除Dom 无

示例

			

            seajs.use('tips', function(Tips){
                
                var tip = new Tips({
                    content:"我就是Tip提示信息"
                    ,zIndex:10009
                    ,width:500
                    ,target:"#btnTip"
                    ,offset:{
                        y:0
                        ,x:0
                    }
                    ,hasCloseBtn:true
                    ,parent:null
                    ,cancel:function(){                     
                        console.log("callback cancel guid:" + this.guid)
                    }
                    ,show:function(){                       
                        console.log("callback show guid:" + this.guid)
                    }
                });
                //tip.setContent(12399);
            });
